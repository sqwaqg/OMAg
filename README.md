## 🚀 Как запустить (по шагам для чайников)

### Шаг 1. Скачай проект
- На GitHub нажми зелёную кнопку **Code** → **Download ZIP** → распакуй куда удобно.
- Или просто открой папку, если она уже у тебя есть.

### Шаг 2. Открой в VS Code
- Запусти VS Code, выбери **File → Open Folder** и укажи папку `OMAg`.

### Шаг 3. Запусти сервер (бэкенд)
> Сервер нужен, чтобы игра помнила твой прогресс. Не закрывай окно, пока играешь.

1. В VS Code открой **новый терминал** (`Terminal → New Terminal`)
2. Введи по очереди:
   ```bash
   cd backend
   npm install        # только в первый раз
   node server.js
Должно появиться: Server running on http://localhost:3001 — всё ок.

Шаг 4. Запусти игру (фронтенд)
Открой второй терминал (кнопка плюсика в VS Code)

Введи:

bash
cd frontend
npm install        # только в первый раз
npm run dev
Появится ссылка http://localhost:5173 — нажми на неё (или скопируй в браузер).

Шаг 5. Играй
Выбирай историю: «Магазин» или «Вклад или кредит».

Читай диалоги (они озвучиваются сами), жми на кнопки.

В мини-играх собирай монетки или распределяй бюджет.

В конце увидишь, что вышло: купил планшет, потерял серёжки, не уложился в бюджет…

📡 Как игра общается с сервером (API)
Метод	Адрес	Зачем
GET	/api/game/state	Узнать текущий баланс и прогресс
POST	/earn	Добавить монетки (например, { "amount": 50 })
POST	/buy	Потратить деньги (например, { "price": 120 })
POST	/nextDay	Перейти к следующему дню
POST	/reset	Начать игру заново
Пример ответа от сервера:

json
{
  "balance": 11500,
  "debt": 0,
  "day": 365,
  "scenario": "deposit",
  "items": ["планшет"]
}
🎮 О чём игры (сценарии)
1. Магазин
У тебя 500 рублей и список обязательных продуктов: хлеб, молоко, яйца, морковь.
Можно купить ещё чипсы или игрушку, но если выйдешь за бюджет — мама расстроится.
Учит не тратить всё на ерунду.

2. Вклад или кредит
Хочешь планшет, но не хватает 2000 рублей.

Мама предлагает вклад: отдай 10000 под 15% годовых. Через год получишь 11500, надо будет докопить 500.

Папа предлагает кредит: даст 2000 сейчас, но через год верни 2300. Если не накопишь — продаст твои любимые серёжки.
Выбирай, потом лови монетки в мини-игре. В конце узнаешь, сохранил серёжки или нет.

🧪 Как проверить, что всё работает
Сервер запустился на http://localhost:3001

Игра открывается на http://localhost:5173

Нажал на кнопку в игре — баланс изменился (посмотри в game-state.json)

Диалоги говорят разными голосами

Кнопка совёнка отключает только его подсказки, а не весь звук

В «Магазине» нельзя купить больше, чем есть денег

В «Ловле монет» монетки засчитываются ровно один раз (раньше было удвоение, но починили)

❓ Что делать, если что-то пошло не так
Проблема	Как решить
npm install ругается	Проверь интернет. Установи Node.js версии 16 или выше
Cannot find module 'express'	Зайди в папку backend и запусти npm install ещё раз
Ошибка CORS	Убедись, что сервер запущен (node server.js) и в коде есть app.use(cors())
Баланс не меняется	Открой game-state.json — если он обновляется, значит API работает. Посмотри консоль браузера (F12)
Нет озвучки	Браузер иногда блокирует звук. Кликни в любом месте игры. Проверь, не выключен ли звук на компе
Порт 3001 или 5173 занят	Закрой другие программы, которые могут их использовать (например, другой сервер)
